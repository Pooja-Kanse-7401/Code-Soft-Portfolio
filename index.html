Deploying a Static Website on AWS

In this project, I have created a static website using HTML and CSS and then deployed it on AWS (Amazon Web Services). Configured a public S3 (Simple Storage Service) bucket to host the website's files and set up CloudFront, AWS's Content Delivery Network (CDN), to enhance the website's performance and content delivery. This project served as an excellent opportunity to learn the basics of AWS and web hosting.
